# Node.js First Class & JavaScript Concepts

## About This Project
This project contains materials and code examples from the first class of Node.js. It is designed to help beginners understand the basics of Node.js and core JavaScript concepts.

## Node.js First Class Topics
- Introduction to Node.js
- Setting up Node.js environment
- Running JavaScript with Node.js
- Understanding the Node.js runtime
- Writing and executing your first Node.js script

## JavaScript Concepts Covered
- Variables and Data Types
- Functions and Scope
- Control Structures (if, for, while)
- Objects and Arrays
- ES6 Features (let, const, arrow functions)
- Asynchronous Programming Basics

---

Feel free to explore the code and experiment with the examples to deepen your understanding of Node.js and JavaScript fundamentals.
